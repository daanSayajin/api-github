Utilização da API

  • URL base da API

    Para todas requisições utilizamos a URL http://api.github.com. A partir dela atribuimos os endpoints que falaremos abaixo

  • Fazer a requisição das informações do usuário

    Utilizamos o endpoint users a partir da URL base citada no 1º passo. Além do endpoint users acrescentamos após ele o endpoint respectivo ao nome do usuário que queremos as informações. Nesse exemplo, resgatamos os dados do usuário daanSayajin https://api.github.com/users/daanSayajin

  • Fazer a requisição dos repositórios do usuário

    Utilizamos o endpoint repos a partir da URL do usuário citada no 2º passo. Nesse exemplo, resgatamos os repositórios do usuário daanSayajin https://api.github.com/users/daanSayajin/repos

  • Fazer a requisição dos repositórios que o usuário atribui uma estrela

    Utilizamos o endpoint starred a partir da URL do usuário citada no 2º passo. Nesse exemplo, resgatamos os repositórios que o usuário daanSayajin atribuiu uma estrela https://api.github.com/users/daanSayajin/starred

  • Fazer a requisição dos seguidores do usuário

    Utilizamos o endpoint followers a partir da URL do usuário citada no 2º passo. Nesse exemplo, resgatamos os seguidores do usuário daanSayajin https://api.github.com/users/daanSayajin/followers

  • Fazer a requisição dos usuários que o usuário segue

    Utilizamos o endpoint following a partir da URL do usuário citada no 2º passo. Nesse exemplo, resgatamos os usuários que o usuário daanSayajin segue https://api.github.com/users/daanSayajin/following
